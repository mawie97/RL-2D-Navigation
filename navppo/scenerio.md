# Curriculum Training Scenerio Generation
| **Level**                                | **Environment Setup & Variation**                                                                                                                                                                                                                                                                                | **Special Features**                                                                                                                                                                                        | **Training Focus / Purpose**                                                                                                                                                                                                                                                                                                                                 |
| ---------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Level 1: Basic**     | **Obstacles:** 0 (none).<br>**Agent-Goal Distance:** Varied (short to long).<br>**Agent & Goal Start:** Random position.                                                                                                                                                                                                                                                                      | Open field with no obstacle.                                                                                                                                     | Learn basic movement and goal-reaching                                                                                                                         |
| **Level 2: Single Obstacle**     | **Obstacles:** 1 static obstacle.<br>**Agent-Goal Distance:** Varied (short to long).<br> **Agent & Goal Start:** Random position.<br>**Obstacle Placement:** Varies in distance from agent (start from far to close to give agent some time to react at first then closer to test reaction).                                                                                                                                                                                                | A single obstacle is placed between the agent and goal. The obstacle might sometimes block the direct path, forcing slight detour.            | Introduce obstacle avoidance and path adjustment around an object. This train **basic avoidance** without overwhelming the agent.                                                                                                                                            |
| **Level 3: Multiple Obstacles**          | **Obstacles:** 3–5 static obstacles.<br>**Agent-Goal Distance:** Varied (short distances use ~3 obstacles; longer distances use up to 5).<br> **Agent & Goal Start:** Random position.<br> **Obstacle Placement:** Obstacles positioned along the direct route between agent and goal (ensuring the straight-line path is blocked).                                                                                                                            | Open area with multiple static obstacles (3–5). Increased obstacle count creates moderate complexity.  | Develop navigation strategies around several obstacles & basic path planning.                                 |
| **Level 4: High-Density Obstacles**      | **Obstacles:** 5–10 static obstacles.<br>**Agent-Goal Distance:** Long range (the goal is placed far from the agent, requiring longer navigation).<br> **Agent & Goal Start:** Random position.<br> **Obstacle Placement:** Spread across the area, with some obstacles likely **on the path** to the goal.                                                                                                          | A heavily cluttered open area. Obstacle **density** is higher, so the agent must navigate a complex, potentially winding route. (Still generally open-area, *not* a narrow corridor yet.)                   | *Advanced obstacle avoidance.* The agent practices **robust navigation** in a highly cluttered environment.                                                                                            |
| **Level 5: Narrow Passages & Dead-Ends** | **Obstacles:** More than 10 and also Variable .<br>**Agent-Goal Distance:** Varied Start from 10 - 30? <br>. **Agent & Goal Start:** Agent or Goal are positioned within the deadend or corridor.<br>**Placement:** Environment includes **corridors or dead-end** sections formed by obstacles. The dead-end is generated in a way the agent must backtrack from it in order to get to the goal. | Constrained spaces introduced. For example, a corridor only slightly wider than the agent (no direct line-of-sight until entering the dead-end). | *Corner-case handling and precision.* The agent learns to **navigate tight spaces** carefully and **back track** out of a dead-end.  This trains more **precise control** and the ability to handle scenarios where the optimal path is not avalible. |
| **Level 6: Mixed / Comprehensive**       | **Obstacles:** Mixed (0 up to 30?, randomly chosen from all the combination of variations above).<br>**Agent-Goal Distance:** Mixed (both short and long instances).<br>**Environment Structure:** Mixed — a random combination of open fields, cluttered fields, and occasional corridor/dead-end setups.                                                                                                                                                     | **Diverse scenarios** presented randomly. One episode might be a simple open field, the next a narrow corridor, etc. This ensures the agent cannot overfit to one type of scenario.                         | *Full generalization training.* By **randomizing all the above factors**, this final stage the agent experience a wide variety of situations. The goal is to test generalization and adaptability to unpredictable, complex scenarios including all prior challenges.         |
